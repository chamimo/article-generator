# GAS セットアップ手順

対象スプレッドシート: ブログ自動生成横展開  
https://docs.google.com/spreadsheets/d/1_pgNf2-JNlT2uwJFGzlVPGpuVpj2mf5eSsa_YLwMwGc/edit

## 1. GAS エディタを開く

スプレッドシート上部メニュー → 「拡張機能」→「Apps Script」

## 2. コードを貼り付ける

`blog_manager.gs` の内容を全コピーして、`コード.gs` に貼り付けて保存（Ctrl+S）。

## 3. onOpen トリガーを設定する

GAS エディタ左側「トリガー（時計アイコン）」→「トリガーを追加」
- 実行する関数: `onOpen`
- イベントのソース: `スプレッドシートから`
- イベントの種類: `起動時`

→ 保存後、スプレッドシートを再読み込みするとメニューバーに「🗂 ブログ管理」が追加される。

## 4. ボタンを設置する（任意）

メニュー「🗂 ブログ管理」→「新規ブログを追加する」でフォーム実行も可能。

ボタンを設置したい場合:
1. GAS エディタで `setupFormButton` を選択して実行
2. 挿入されたボタン図形を右クリック → 「テキストを編集」→「ブログを追加する」と入力

## 5. 動作確認

「新規ブログ追加」シートの B2〜B8 に値を入力し、
メニュー「🗂 ブログ管理 > 新規ブログを追加する」を実行。
→ 「ブログ管理」シートに1行追加・ステータス「準備中」で登録される。
