# protectguardfilm.com WordPress初期設定メモ

## 前提

- 対象サイト: https://protectguardfilm.com
- サイト種別: 法人向け特注保護フィルムの紹介・相談窓口
- 投稿先として `groowill.co.jp` は使わない
- 本番ドメインとして `exactfitfilm.com` は使わない
- 企業様の公式サイトではなく、当方が管理・運営する相談窓口として表現する
- 記事生成と投稿は、WordPress側の受け皿が整ってから再開する
- 投稿ステータスは当面 `draft` のまま運用する

## 接続設定

`.env` またはサイト固有 `.env` に以下を設定する。

```env
GROOWILL_FILM_WP_URL=https://protectguardfilm.com
GROOWILL_FILM_WP_USERNAME=
GROOWILL_FILM_WP_APP_PASSWORD=
GROOWILL_FILM_WP_STATUS=draft
GROOWILL_FILM_WP_CATEGORY_ID=1
GROOWILL_FILM_GSC_SITE_URL=https://protectguardfilm.com/
```

`blogs/groowill-film/blog_config.json` の `wp_url` も `https://protectguardfilm.com` を基準にする。

## プライバシーポリシー・フォーム文言案

固定ページ更新前に、以下の趣旨をプライバシーポリシーとお問い合わせフォーム付近へ入れる。

- お問い合わせ内容は、仕様確認・お見積り・納期確認・問い合わせ対応のため、当サイト運営者および提携先企業に共有される場合がある
- 提携先企業から、メールまたは電話で直接連絡する場合がある
- 共有対象には、会社名、担当者名、連絡先、端末名、型番、導入台数、希望機能、写真・図面、希望納期、相談内容が含まれる場合がある
- 取得した情報は、問い合わせ対応、仕様確認、お見積り、納期確認、関連する連絡の目的で利用する

## 投稿再開前チェック

- `https://protectguardfilm.com/wp-json/wp/v2` が応答する
- WordPressアプリケーションパスワードを発行済み
- カテゴリを作成済み
- お問い合わせフォームを設置済み
- 固定ページを作成済み
- `GROOWILL_FILM_WP_URL` が `https://protectguardfilm.com` になっている
- `GROOWILL_FILM_WP_STATUS` が `draft` になっている
- dry-runでHTML/JSONを確認済み
- 本番投稿は明示確認後に実行する
